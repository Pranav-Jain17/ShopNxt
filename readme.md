# ShopNxt 🛒

A lightweight, responsive e-commerce frontend built entirely with Vanilla web technologies. This project fetches mock product data to demonstrate core frontend concepts like state management, DOM manipulation, and asynchronous API calls—all without the need for heavy frameworks.

## ✨ Features
* **Product Catalog:** Responsive grid layout with client-side pagination.
* **Product Details:** Dedicated view for individual product information.
* **Shopping Cart:** Add items, track quantities, and automatically calculate subtotals and taxes.
* **Smooth UI:** Custom CSS keyframe animations, hover states, and interactive user feedback.
* **Zero Dependencies:** No `npm install` or build steps required.

## 🛠️ Tech Stack
* **HTML5**
* **CSS3** (Flexbox, CSS Grid, custom animations)
* **Vanilla JavaScript** (ES6+, Async/Await)
* **[DummyJSON API](https://dummyjson.com/)** (Mock backend data)

## 🚀 How to Run
Running this project is incredibly simple:
1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.
3. Start shopping!
